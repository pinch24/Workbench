# 01. 언리얼 엔진 소개

## 개요
게임 엔진은 비디오 게임을 처음부터 만드는 데 필요한 기능을 제공하는 애플리케이션이다. 게임 엔진이 제공하는 기능들은 상당히 다양하지만, 일반적으로 3D 모델, 이미지, 소리, 비디오와 같은 멀티미디어 파일을 임포트할 수 있는 기능을 제공하며, C\++, C#와 같은 프로그래밍 언어를 사용할 수 있는 프로그래밍 환경에서 이런 파일들을 조작할 수 있는 기능을 제공한다.
